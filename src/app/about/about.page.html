Abu Dhabi Ports Pass is a mobile application which allows anyone who wants to enter Abu Dhabi ports to apply for gate pass. One can either register themselves or if registered already, apply for gate pass using this mobile application. This will automatically submit the gate pass application to the next level of approval and customers will be notified once the application has been approved. The mobile application will show status of the gate pass at any point in time and will also show analytics report for all passes applied till date.
